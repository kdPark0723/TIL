### 탭 컨트롤

- 탭 추가


  ```c++
  int CTabCtrl::InsertColunm(int nItem, LPCSTR lpszItem);
  ```

  ```c++

  ```

  ```c++
  m_tabSelection.GetCurSel();
  ```

  ```c++
  int CTabCtrl::GetCurSel();
  ```
